Master of Information Technology

Software Development Major
Queensland University of Technology, Brisbane

Programming has always been a hobby of mine. I first developed my passion in high school where I taught myself to use Java. I started tinkering with the source code released for a game, adding in my own improvements while using online resources for direction. This was where I first discovered the potential and power of programming.

I kept this hobby up throughout my Bachelor of Engineering, and shortly after graduating, made the decision to commence this course. Over the duration of this degree, I have gained valuable experience using C#, Javascript and Python. I have also been introduced to the React and React Native framework library through my Stock Blast project. On top of this, I have also been taught how to create AI and machine learning solutions, and how to use Docker containers and Amazon Web Services.

This course has given me a solid grounding in a range of programming languages for different applications. I have learnt some fundamental data structures and algorithms, as well as how to assess program complexity. I have also learnt how to build solutions from the ground up. The course has given me the confidence to perform in any software development environment by not just teaching theory, but having students complete relevant development projects. I have been equipped with critical thinking and problem solving skills that I can put to use in the technology industry.
