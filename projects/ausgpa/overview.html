Australian GPA Calculator

Calculate your current or required GPA of the semster's assessment.

Australian GPA Calculator was created for submission as my final project for Harvard's online CS50x course. The requirement was to create something that changed the world, benefitted your community or meant something to you. I identified that each semester I would perform a series of GPA calculations on a piece of paper to determine possible outcomes and requirements. Ultimately this wasted time as the set of calculations would have to be remembered and repeated each semester. This project aimed to simplify and automate the calculations for myself and all Australian students using the 7-point GPA scale.

The website was created as barebones as possible to prioritise performance. For example, it is purely front-end, with no server. A server was not possible without paying for a webserver and GitHub Pages only hosts html pages for free. This did not end up being a limitation to the project as the concept only required calculations to be performed in the browser.

One of my goals was for the website to be interactive with a smooth user experience. I utilised Bootstrap to give a consistent feel across the page and created a neutral but pleasing seamless tile for the background. The final product incorporates Javascript animations to expand and hide fields as required by the input given by the user.
