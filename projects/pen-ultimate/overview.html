Pen Ultimate

eCommerce site for pens created using Flask.

This website was created as the final submission to the QUT unit IFN557 Rapid Web Development. The requirement of this unit was to create an eCommerce website based around any category of product. To avoid any issues regarding copyright, I decided to use my own items as the products. After a quick scan of my room, I realised I had a moderate amount of pens which would suit well. I got creative and used a sheet of paper for the pen backdrops to give it a more consistent and professional look. As a quick side-note, if it wasn't obvious enough, after realising I didn't have any red clicker pens, I digitally edited the blue clicker pen to be red.

The greatest challenge I faced during this project was when converting from a dummy-data website to a database-driven website. Since I already had a code base to work with, this required me to convert all functions at once which made debugging extremely difficult. The reason this approach was taken was because databases, or the model in MVC, was taught last in the unit. The lesson I took away from this difficulty was to gather product requirements early and ensure code functionality is only minimally changed at a time.

This unit fully explored the concepts of the Model-View-Controller (MVC) design pattern. Through it I learnt the importance of separation of concerns such that a user does not directly interact with the data. This allows for greater data integrity and security, as well as transformation operations to represent the data in multiple ways depending on the View context.
